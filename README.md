## Setup & Configure BOT

### Linux
1. Clone project repo
   ```
   git clone https://github.com/Widiskel/gaia-chat-bot.git && cd gaia-chat-bot
   ```
2. Run
   ```
   npm install 
   npm run setup
   ```
3. Configure config file using `nano` or your `text editor`
   ```
   nano config/config.js
   ```
4. Run Bot
   ```
   npm run start
   ```
   
## Update Bot

To update bot follow this step :
1. run
   ```
   git pull
   ```
   or
   ```
   git pull --rebase
   ```
   if error run
   ```
   git stash && git pull
   ```
2. run
   ```
   npm update
   ```
2. start the bot
